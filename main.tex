\title{\large\bfseries 4D-MLLM: Learning a 4D Latent World Model from 2D Video \\ via Structured SpatioTemporal Transformers}

\begin{abstract}
Video-LLMs have advanced rapidly, yet most still treat video as a sequence of 2D patch tokens with time represented implicitly by frame order. Recent benchmarks show that this representation is brittle for spatiotemporal reasoning and physical understanding in dynamic scenes \cite{zhou2025vlm4d, zhu20254dbench, wang2025spatial4dbench}. We propose 4D-MLLM, a new backbone paradigm that learns a unified 4D latent world representation directly from timestamped 2D video. The core unit is a 4D latent token $T_i = (f_i, x_i, \tau_i)$, where $f_i$ is appearance, $x_i$ is a learned latent 3D coordinate, and $\tau_i$ is absolute time. By inducing geometry through self-supervised consistency and using structured spatiotemporal attention, 4D-MLLM targets joint 3D and time reasoning without requiring explicit SLAM, depth sensors, or full 4D reconstruction.
\end{abstract}

\section{Introduction}
\label{sec:introduction}

The current situation in video-language modeling is paradoxical. Video-LLMs can generate fluent descriptions and answer many semantic questions from RGB video \cite{zhang2023video, damonl2024videollama2, li2024llavanext}. However, when questions require stable 3D structure over time, such as tracking objects through occlusion, comparing motion under viewpoint changes, or maintaining correct temporal causality, performance degrades sharply. This gap is now quantified by spatiotemporal benchmarks, which reveal substantial deficits in temporal coherence, viewpoint-aware motion understanding, and physical plausibility \cite{zhou2025vlm4d, zhu20254dbench, wang2025spatial4dbench}.

In parallel, early 4D MLLMs and 4D vision-language systems have begun to incorporate explicit spatiotemporal structure. Uni4D-LLM is an early attempt to unify 4D understanding and 4D generation by injecting spatiotemporal geometric cues into a shared Transformer representation \cite{zhou2025uni4dllm}. Other work demonstrates strong 4D modeling when explicit dynamic 3D inputs are available, such as dynamic point clouds or LiDAR sequences \cite{4dpc2hat2026, b4dl2025}. Complementary directions include language grounding in dynamic 4D fields \cite{4dlangsplat2025}, 4D-aware vision-language-action models for robotics \cite{4dvla2025}, and region-level 4D perception via distillation \cite{yang20254drgpt}.

These trends point to a central conclusion: the bottleneck is not a missing ``module'', but the backbone representation itself. This proposal targets the missing backbone paradigm for timestamped 2D video: a model that (1) changes the token definition from 2D patches to 4D entities, (2) changes attention to be biased by spatiotemporal proximity, and (3) changes the representation target from ``video features'' to a unified 4D latent world representation. We introduce 4D-MLLM, a 4D latent world model transformer designed around these three changes, aiming to bridge semantic video understanding and structured 3D plus time reasoning using ubiquitous 2D video inputs.

\section{Proposed Method}
\label{sec:methodology}

We propose a backbone that transforms the fundamental unit of computation from a ``2D patch'' to a ``4D token''. Concretely, for each token we maintain a tuple $T_i = (f_i, x_i, \tau_i)$, where $f_i$ is the visual feature (appearance), $x_i$ is a learned latent 3D coordinate that organizes tokens into a geometry-consistent space, and $\tau_i$ is the absolute timestamp associated with the observation. The key point is that $x_i$ is not ground-truth geometry and does not rely on explicit reconstruction; it is learned as part of the representation target so that the backbone naturally supports questions of ``what is where, when'' and can maintain object permanence across views and time.

To learn $x_i$ from timestamped 2D video, we induce geometry through self-supervised spatiotemporal consistency rather than SLAM-style pipelines. The model is trained to align tokens that correspond to the same physical entity across time and viewpoint changes, enforcing stability for static regions ($x_{i,t} \approx x_{j,t+1}$) while allowing structured evolution for dynamic regions. Time is modeled as a continuous variable using learnable Fourier features to handle variable framerates and precise durations. Importantly, the model learns absolute time understanding by encoding real-world timestamps directly in the representation, which enables consistent reasoning over long videos, irregular sampling, and multi-camera synchronization. Dynamics are represented with a time-conditioned predictor that relates changes in $x$ to elapsed time, encouraging temporal scaling consistency.

On top of these 4D tokens, we modify attention to reflect spatiotemporal structure: attention is biased so that tokens that are close in latent 3D space and close in time interact more strongly than distant ones, supporting locality and causal propagation in the learned world model. We train the backbone with Masked 4D Modeling by masking a spatial region (in latent space) or a temporal interval and requiring reconstruction from the remaining context, which encourages both occlusion reasoning and trajectory completion. Evaluation treats 4D-MLLM as a backbone rather than a task-specific head, using benchmarks that expose 4D reasoning gaps including VLM4D \cite{zhou2025vlm4d}, 4D-Bench \cite{zhu20254dbench}, and Spatial4D-Bench \cite{wang2025spatial4dbench}, with comparisons to strong video-language baselines \cite{damonl2024videollama2, li2024llavanext} and ablations over token definition, attention bias, and representation objectives.

\section{Expected Outcomes}
\label{sec:outcomes}

We expect 4D-MLLM to deliver a backbone that moves beyond sequence modeling and produces a stable, evolving 4D latent world representation that is useful across downstream tasks. In practice, this should translate into stronger 3D-aware question answering (e.g., resolving relative depth and spatial relations under viewpoint changes), more reliable metrology-style reasoning (e.g., inferring speed and trajectory from changes in latent position over absolute time), and improved occlusion reasoning through object permanence, where entities remain represented even when temporarily invisible in the raw video stream.
