\begin{abstract}
Current Video-LLMs treat video as a sequence of 2D patches, effectively reducing "time" to a frame index and "space" to flattened features. This sequence modeling approach lacks explicit 3D structure and continuous temporal reasoning, preventing models from understanding the physical "world" behind the pixels. We propose 4D-MLLM, a paradigm shift from video sequence modeling to 4D World Modeling. Instead of processing 2D patches, 4D-MLLM learns 4D Latent Tokens comprising visual features, learned latent 3D coordinates, and continuous timestamps. By employing a Geometry Induction Head to learn 3D consistency from self-supervision and a Structured 4D Attention mechanism with spatio-temporal biases, the model constructs a unified 4D latent representation. Trained via Masked 4D Modeling, 4D-MLLM enables true joint reasoning over space, time, and object dynamics without requiring explicit SLAM or depth supervision.
\end{abstract}

\section{Introduction: The Crisis of Representation in Video-LLMs}
\label{sec:introduction}

The field of Video Understanding is currently bifurcated. On one side, Video-LLMs like Video-LLaMA 2 \cite{damonl2024videollama2} and LLaVA-NeXT-Video \cite{li2024llavanext} excel at high-level semantic reasoning by treating video as a sequence of 2D visual tokens. However, they fundamentally lack a 3D world model; "space" is flattened into feature vectors, and "time" is reduced to discrete positional indices. This results in a critical failure mode: they hallucinate physical dynamics, struggle with occlusion permanence, and cannot perform precise metrology \cite{liu2024spatial}.

On the other side, Native 4D Models like 4DPC²hat \cite{4dpc2hat2026} and B4DL \cite{b4dl2025} explicitly process 3D point cloud sequences or LiDAR data.

This proposal identifies a third, missing path: Inducing a 4D World Model from 2D Video. We argue that we do not need explicit 3D sensors to learn 4D structure. By enforcing geometric consistency and continuous temporal dynamics within the latent space of a Transformer, we can construct a "4D-MLLM" that combines the semantic richness of Video-LLMs with the physical grounding of Native 4D models.

\section{Related Work: A Fractured Landscape}
\label{sec:related}

\textbf{Emerging 4D MLLMs.} The landscape has shifted rapidly in 2025-2026. \textbf{Uni4D-LLM} \cite{uni4d2025} stands out as a unified framework for 4D understanding and generation, fusing spatiotemporal representations via adaptive cross-attention. \textbf{4DPC²hat} \cite{4dpc2hat2026} targets dynamic point clouds, proving the viability of MLLMs for native 4D data. Other efforts like \textbf{4D-RGPT} \cite{4drgpt2025} explore region-level 4D distillation. Our 4D-MLLM differs by inducing this structure directly from standard RGB video, making it a more universal backbone.

\textbf{Benchmarks Driving Development.} New benchmarks have exposed the limitations of current models. \textbf{4D-Bench} \cite{4dbench2025} and \textbf{Spatial4D-Bench} \cite{spatial4dbench2026} specifically evaluate 4D object understanding and spatial intelligence. Results show that even advanced MLLMs struggle with temporal tracking and physical reasoning, underscoring the need for the structured 4D-MLLM approach we propose.

\textbf{World Models.} The concept of learning a "World Simulator" \cite{yang2024world} is gaining traction. 4D-MLLM aligns with this vision but focuses on *latent* world modeling—constructing a structured internal representation $Z$ that obeys physical laws, rather than generating pixels.
\label{sec:methodology}

We propose a backbone that transforms the fundamental unit of computation from a "2D Patch" to a "4D Token".

\subsection{Core Design: 4D Tokenization}
Unlike traditional ViTs that map patches to simple embeddings, we define a 4D Token $T_i$ as a tuple combining visual, geometric, and temporal attributes: $T_i = (f_i, x_i, \tau_i)$. Here, $f_i$ represents the visual feature vector (appearance), $x_i$ denotes the learned latent 3D coordinate (geometry), and $\tau_i$ encodes absolute continuous time (dynamics). Crucially, $x_i$ is not a ground-truth 3D point but a learned embedding in a latent metric space, allowing the model to self-organize spatial relationships without explicit supervision.

\subsection{Geometry Induction: Learning 3D without SLAM}
We avoid expensive explicit 3D reconstruction methods like SLAM or NeRF. Instead, we introduce a Geometry Induction Head trained via self-supervision. The core principle is that if visual patches across time undergo appearance changes consistent with perspective shifts, they must share the same underlying 3D latent coordinate $x$. The model predicts this latent coordinate $x_i = g(f_i, \text{context})$ and is constrained by a 3D Consistency Loss: for tokens identifying the same static object, the predicted 3D coordinates should remain stable ($x_{i,t} \approx x_{j,t+1}$), while dynamic objects must follow learned kinematic laws. This mechanism forces the model to organize its latent space geometrically.

\subsection{Continuous Time \& Dynamics Modeling}
Time is modeled as a continuous variable using learnable Fourier features, enabling the system to handle variable framerates and precise durations. Unlike traditional methods that treat time as relative frame indices, our model learns **Absolute Time Understanding**, encoding real-world timestamps (e.g., specific seconds or hours) directly into the latent representation. This allows the model to ground events in a concrete temporal reality, essential for correlating disparate events across long videos or multi-camera setups. We explicitly model dynamics via a Time-Aware Dynamics Head that learns to predict how the latent position $x$ of an object evolves over a time interval $\Delta \tau$. This module effectively internalizes velocity and motion constraints, allowing the model to understand not just where an object is, but how it is moving in absolute terms.

\subsection{Structured 4D Attention Mechanism}
Standard Self-Attention is geometry-agnostic, treating all tokens as equidistant. We propose a 4D Attention mechanism with explicit Spatio-Temporal Biases. The attention score is modulated by terms that penalize attention between physically distant tokens (in latent 3D space) and temporally distant events. By incorporating learnable parameters for these biases, the model naturally learns to prioritize information that is "spatially near" and "temporally causal," mimicking physical interactions in the real world.

\subsection{Training Objective: Masked 4D Modeling}
To train this backbone as a foundation model, we employ **Masked 4D Modeling**:
\begin{itemize}
    \item We randomly mask a **spatial volume** (e.g., a region in latent 3D space) or a **temporal interval**.
    \item The model must reconstruct the missing tokens ($f, x$) using the spatiotemporal context. This forces it to learn object permanence (recovering occluded objects) and trajectory prediction (filling missing time steps).
\end{itemize}

\section{Expected Outcomes & Distinction}
\label{sec:outcomes}

\textbf{Paradigm Shift:} 4D-MLLM moves from "Sequence Modeling" to "World Modeling". It does not just predict the next word; it maintains a consistent, evolving 4D representation of the scene.

\textbf{Capabilities:}
\begin{itemize}
    \item \textbf{3D-Aware QA:} "Is object A behind object B?" (solved via latent $x$ comparison).
    \item \textbf{True Metrology:} "What is the speed?" (solved via $\Delta x / \Delta \tau$).
    \item \textbf{Occlusion Reasoning:} Understanding that an object still exists at coordinate $x$ even when visual features $f$ are occluded.
\end{itemize}

\section{Related Work}
\label{sec:related}

This proposal diverges from standard Video-LLMs like Video-LLaMA \cite{zhang2023video}, which lack explicit geometry. Unlike Native 4D models \cite{4dpc2hat2026} that require point cloud inputs, 4D-MLLM \textit{induces} geometry from 2D video. It aligns with the ``World Model'' vision but applies it specifically to constructing a structured backbone for Multimodal LLMs.
